# inform-parking
주차장 위치와 남은 공간, 가격 등을 제공하는 웹프로젝트

* Front : jsp, html, css, java-script, bootstrap, apache tile
* Server : Apache, Tomcat, Docker
* DataBase : MyBatis, MySQL, MySQLWorkBench
* IDE : intellij
* Framework : Spring MVC - Maven사용
